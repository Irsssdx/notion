MongoDB Query

ü•Ç MongoDB Insert Query -------- insertOne()  /  insertMany() 

User.insertOne({ name: "Company Inc", address: "Highway 37" });

User.insertMany([
    { name: 'John', address: 'Highway 71'},
    { name: 'Peter', address: 'Lowstreet 4'},
    { name: 'Amy', address: 'Apple st 652'},
]);

ü•Ç MongoDB Read Query -------- find()  /  findOne() 

User.find();

User.findOne({ _id: id });

// PROJECTION --- means which fields you want to include
// 0 = exclude (it will not display) & 1 = include

User.find({}, { projection: { _id: 0, name: 1, address: 1 });
User.findOne({ _id: id }, { projection: { _id: 0, name: 1, address: 1 });

// OUTPUT
[
	{ name: 'John', address: 'Highway 71'},
	{ name: 'Peter', address: 'Lowstreet 4'},
	{ name: 'Amy', address: 'Apple st 652'},
]

ü•Ç MongoDB SORTING -------- .sort() 

User.find().sort({ name: 1 });

// 1 means sort alphabetically a-z in ascending order & -1 means z-a in descending order
